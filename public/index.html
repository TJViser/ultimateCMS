How it works

1
Click any text on the live website to edit it inline. Modified elements get a green indicator.
2
Submit your changes. The old text, new text, and full context (CSS classes, DOM path, section, URL) are sent to the backend.
3
AI agent analyzes the codebase. It greps the repo, reads candidate files, and uses Claude to find the exact location in the source — HTML, JSON, Markdown, JSX, i18n files, templates, whatever.
4
Pull request created. The agent edits the source file(s), creates a branch, and opens a PR. Review, approve, merge.
